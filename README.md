# Prismagram

Instagram clone with Express + Prisma + React + React Native

## User Stories

- [x] Create account
- [x] Log in
  - [x] Request Secret
  - [x] Confirm Secret
- [x] Like / Unlike a photo
- [x] Comment on a photo
- [x] Search User
- [x] Search Post
- [ ] See user profile
- [x] Follow / Unfollow User
- [ ] See the full photo
- [ ] Edit my profile
- [ ] Upload a photo
- [ ] Edit the photo (Delete)
- [ ] See the feed
